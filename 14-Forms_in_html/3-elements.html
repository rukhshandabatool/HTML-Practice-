Label tag

(A <label> is an html tag used to name or describe a form input, like a text box or checkbox, so users know what to enter.)



EXTRA INFO

~ It tells the user: “This box is for your name” or “Click here to agree, make sure that all the id's are unique in a page”.

~ Now if you click “Your Name:”, the typing box becomes active — easy for everyone to use, even on mobile or with screen readers.

~ Make sure that for = "" & id = "" value should same to make Your Name clickable.
